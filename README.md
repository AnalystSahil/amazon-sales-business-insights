# Amazon Sales SQL Analysis Project

## Project Overview
This project analyzes Amazon sales data using SQL. The goal is to extract meaningful business insights like revenue trends, product performance, customer behavior, and payment analysis.

## Dataset Information
- Dataset: Amazon Sales Dataset (CSV format)
- Contains: order details, product category, revenue, ratings, region, payment methods

## Tools Used
- MySQL
- SQL (Aggregations, Window Functions, Case Statements)

## Key Analysis Performed
- Total revenue calculation
- Category-wise sales performance
- Top and bottom performing categories
- Payment method analysis
- Customer region analysis
- Rating-based segmentation
- Monthly and yearly sales trends
- Revenue ranking using window functions
- Discount impact analysis

## Key Business Insights
- Identified top revenue-generating categories
- Found most used payment methods
- Discovered high and low performing product segments
- Analyzed customer behavior across regions
- Compared categories based on revenue and quantity

## Files in Project
- amazon_sales_analysis.sql → All SQL queries
- amazon_sales_dataset.csv → Dataset used for analysis
